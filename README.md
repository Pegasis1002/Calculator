
# Compile instructions
run the build.py file to compile the program
## 1. Linux
In the root directory of the project
  ./build.py

## 2. Windows
In the root directory of the project
```bash
  python build.py
```

# Run program
## 1. Linux
In the root directory of the project
```bash
  ./build/calculator
```

## 2. Windows
In the root directory of the project
```bash
  .\build\debug\calculator.exe
```
